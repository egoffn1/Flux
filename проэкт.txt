## 5. КЛЮЧЕВЫЕ КОМПОНЕНТЫ

### 5.1 MusicService.kt
   - @AndroidEntryPoint MediaSessionService
   - Инжектит: QueueManager, AudioEqualizer, FavoritesRepository
   - ExoPlayer + MediaSession
   - Уведомление (onUpdateNotification):
     * 6 actions: Previous, Play/Pause, Next, -10s, +10s, Like (звёздочка)
     * Compact: 3 кнопки (Prev, Play/Pause, Next)
     * Обложка через MediaMetadataRetriever (IO thread) + кэш per-track
     * Like-статус через FavoritesRepository
   - handleCustomAction: ACTION_REWIND / ACTION_FORWARD / ACTION_LIKE
   - startForeground при каждом обновлении (album art loaded, like toggled)

### 5.2 AudioEqualizer.kt
   - @Singleton, injectable
   - 10 прессетов: Normal, Classical, Dance, Flat, Folk, Heavy Metal, Hip Hop, Jazz, Pop, Rock
   - PRESET_BANDS: shortArray с 8 полосами (-600..600)
   - attachToSession(sessionId), applyPreset(preset), setBassBoost(level)
   - setBandLevel(band, level), getBandLevels(), getCenterFrequencies() — для band sliders
   - LoudnessEnhancer для Bass Boost

### 5.3 NowPlayingScreen.kt
   - Полноэкранный плеер
   - Автоматическое определение ориентации (LocalConfiguration.current.orientation)
   - Portrait: Column (AlbumArt → TrackInfo → WaveProgressBar → Controls)
   - Landscape: Row (AlbumArt 40% слева, Controls 60% справа)
   - Анимации: pulseScale, iconPlayFraction, iconGlowPhase, playButtonScale, skipScale, favoriteTint
   - AlbumArtCard с AmbientWaveBackground + WaveVisualizer + AnimatedContent
   - ControlsSection: Shuffle, Previous, Play/Pause, Next, Play Similar, Repeat
   - WaveProgressBar с перемоткой и formatDuration
   - ModalBottomSheet для добавления в плейлист

### 5.4 Theme.kt
   - FluxTheme(themeMode, dynamicColor)
   - themeMode: 0=System, 1=Light, 2=Dark
   - dynamicColor: true=Material You (Android 12+), false=кастомная палитра
   - LightColorScheme / DarkColorScheme (Material 3)
   - SideEffect: statusBarColor, navigationBarColor transparent
   - Читается из UserPreferences в MainActivity

### 5.5 WaveVisualizer.kt
   - vsync-синхронизированная анимация через LaunchedEffect + withFrameMillis
   - mutableFloatStateOf для phase (0..2π) и smoothPhase
   - animateFloatAsState для playFraction (400ms, FastOutSlowInEasing)
   - 4 типа волны: BARS, SINE, CIRCULAR, GRADIENT_BARS
   - CompositionLocals: LocalWaveType, LocalPlayingTrackId
   - TrackItem читает BeatState.beatIntensity для пульсации

### 5.6 FluxApp.kt
   - Coil ImageLoader с:
     * MemoryCache: 25% of heap
     * DiskCache: 50MB, cacheDir/coil_cache
     * crossfade: true

### 5.7 SettingsScreen.kt
   - Sound: Equalizer (вкл/выкл + presets + band sliders с частотами + bass boost)
   - Wave Visualization: вкл/выкл + style + bar count + speed
   - Theme: Dynamic Colors on/off + Theme Mode (System/Light/Dark)
   - Updates: auto-check + check now
   - About: version, GitHub link

## 9. ВАЖНЫЕ ЗАМЕЧАНИЯ
   - TrackItem не требует isPlaying от родителя — определяет сам через LocalPlayingTrackId
   - BeatState — Kotlin object, доступен отовсюду без DI
   - QueueManager — @Singleton, injectable через Hilt
   - MediaSessionConnection — @Singleton, injectable
   - Все экраны используют hiltViewModel()
   - LibraryViewModel, PlaylistsViewModel имеют QueueManager инжектированный
   - SearchViewModel НЕ имеет QueueManager
   - Canvas в TrackItem использует 56dp размер с RoundedCornerShape(12dp)
   - playFraction анимируется 300ms с FastOutSlowInEasing
   - WaveVisualizer: vsync через withFrameMillis, mutableFloatStateOf
   - AudioEqualizer: @Singleton, 10 пресетов с 8 полосами
   - Уведомление: onUpdateNotification, 6 actions, like через FavoritesRepository
   - Тема: System/Light/Dark, Dynamic Colors on/off, сохранение в DataStore
   - Ландшафт: Row с AlbumArt 40% + Controls 60%
   - Coil: DiskCache 50MB, MemoryCache 25%
